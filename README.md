# Password Manager 🔐

A simple command-line Password Manager built using Python.

## 📌 Features
- Store passwords securely in a file
- Retrieve saved passwords
- Simple and easy-to-use interface

## 🛠️ Tech Used
- Python
- File Handling

## 🎯 Purpose
This project was built to practice Python programming and understand file handling concepts.

## 🚀 Future Improvements
- Add encryption for passwords
- Add GUI version
